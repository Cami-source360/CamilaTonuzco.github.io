Project 02

Differential Drive Underwater Robot (ROV)

Design, construction, programming, and testing of a remotely operated differential-drive aquatic robot

Project Details

Completed: November 2025

Project Overview

This project focused on the design, construction, programming, and validation of a Remotely Operated Vehicle (ROV) using a differential thrust configuration for underwater mobility.

The objective was to develop a low-cost but robust underwater platform capable of stable navigation, controlled buoyancy, and real-time teleoperation.

The system was developed iteratively, progressively improving mechanical stability, depth control, and overall system reliability. The final platform integrates propulsion, buoyancy actuation, sensing, and a ROS2-based control architecture.

Main capabilities include:

  • Differential thrust control for forward, reverse, and turning motion
  • Active buoyancy system for depth regulation
  • Real-time sensor feedback for state monitoring
  • Joystick-based teleoperation via ROS2 communication

Mechanical Design

ROV Structure

The mechanical body is based on a 6-inch PVC cylindrical housing, chosen for its durability, availability, and ease of manufacturing. Internal components were built using acrylic, MDF, and 3D-printed parts to enable modular assembly and fast iteration.

Key mechanical elements include:

  • Propulsion system: two rear DC motors with propellers enabling differential thrust control
  • Buoyancy system: syringe-based mechanism driven by a stepper motor and leadscrew for depth control
  • Internal structure: laser-cut mounting plates for electronics and subsystems
  • Sealing system: O-rings and cable glands for waterproofing
  • Modular housing: removable top cover for maintenance and testing

The design ensures stability by balancing center of mass and buoyancy for neutral underwater behavior.

Control System

The ROV is controlled using a ROS2-based architecture enabling real-time teleoperation and structured communication between user input and onboard systems.

  • Receives commands through /cmd_vel_joy_modified ROS2 topic
  • Joystick interface maps user inputs to propulsion and buoyancy control
  • Object-oriented firmware handles DC and stepper motor actuation
  • Periodic control loop manages timing and system coordination

Control logic includes:

  • Differential thrust for steering and forward motion
  • Step-based buoyancy control for depth adjustment
  • Time-safe actuation to avoid sudden mechanical stress

Electronics

The electronic architecture integrates sensing, actuation, and communication into a compact embedded system.

Main components:

  • Raspberry Pi Pico microcontroller
  • TB6612FNG motor drivers
  • PC851 optocoupler isolation
  • MP1484 buck converter
  • MPU-6050 IMU sensor
  • MPS20N0040D pressure sensor
  • TEMT6000 light sensor

Actuation system:

  • 2 DC propulsion motors
  • 1 stepper motor for buoyancy control

The system communicates via ROS2 using micro-ROS integration for real-time embedded control.

Testing & Validation

  • Verification of differential thrust motion (forward, reverse, rotation)
  • Validation of buoyancy system for controlled ascent/descent
  • Waterproofing integrity tests
  • Sensor validation for depth and environmental awareness
  • Safety and stability checks under continuous operation

The final prototype demonstrated stable underwater navigation, reliable buoyancy control, and consistent joystick-based operation.
